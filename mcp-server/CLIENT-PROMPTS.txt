═══════════════════════════════════════════════════════════════════
  DEPLOYMENT STEPS REMINDER
═══════════════════════════════════════════════════════════════════

1. Get API Token from PelangiManager Settings → Security
2. Create Zeabur service (root: mcp-server)
3. Set environment variables:
   - PELANGI_API_URL=https://pelangi.zeabur.app
   - PELANGI_API_TOKEN=<your-token>
   - MCP_SERVER_PORT=3001
   - NODE_ENV=production
4. Configure domain (e.g., mcp-pelangi.zeabur.app)
5. Deploy and verify
6. Update YOUR_DEPLOYED_URL in prompts above
7. Configure each MCP client
8. Test with verification prompts
